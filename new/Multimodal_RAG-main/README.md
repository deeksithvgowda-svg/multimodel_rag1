# Multimodal RAG with Jina Embeddings v4 + Groq

This project implements a multimodal Retrieval-Augmented Generation pipeline:

- Text file ingestion
- Image OCR extraction
- Jina Embeddings v4 (API-based)
- FAISS similarity retrieval
- Groq Llama Instant answering
- Streamlit user interface

## API Key Options

- Enter `Groq API Key` directly in the sidebar, or
- Enable `Use Groq key from environment (GROQ_API_KEY)` to load it from your environment variable.
- Enter `Jina API Key` directly in the sidebar, or
- Enable `Use Jina key from environment (JINA_API_KEY)` to load it from your environment variable.
